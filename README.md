# UBC sublet finder

https://subletify.streamlit.app/

## Tools Used:
Python: pandas, numpy

Streamlit

## About:
Parser to read WhatsApp subletting group chat messages into a csv file + streamlit website to display listings that fulfill a set of user inputted filters. 
The website helps streamline the process of looking for a particular type of sublet, replacing the inefficiency of having to scroll through a group chat. 
Filtering options include:
- Rent
- Residence
- Unit type
- Gender
- Date posted
- Negotiability

Made primarily to strengthen familiarity with pandas.

To run locally:

pip install -r requirements.txt

streamlit run website.py
